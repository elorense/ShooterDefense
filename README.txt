ShooterDefense:

This is a Tower defense style shooter game that I made using XNA Visual Studio in C#. 
The goal of this game is to shoot down all the monsters that are trying to take the treasure from you.
There is a two player mode. 


Weapons:
Basic Weapon: 10dmg
Second Weapon: 30dmg
Third Weapon: 15dmg Area-of-effect

XBOX CONTROLS:
A - Cycle through weapons
Right shoulder - use weapon
Left stick - move cursor

KEYBOARD CONTROLS:
P1:
W - Up
S - Down
A - Left
D - Right
Space - Use Weapon
Tab - Cycle through weapons

P2:
Arrow Up - Up
Arrow Down - Down
Arrow Left - Left
Arrow Right - Right
Enter - Use Weapon
Right shift - Cycle through weapons
